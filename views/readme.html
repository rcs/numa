Numa - A Swiss Army HTTP Server

Test all kinds of HTTP requests and responses.

Freely hosted at Heroku

Endpoints

Grab Bag

/ This page

/ip Returns Origin IP

/html Returns an HTML rendered page

/stream/:n Returns :number streaming (Transfer-Encoding: chunked) response, with a delay between each specified in ?delay

/status/:codes Returns a randomly selected status code from :codes

/delay/:n Returns basic request information after :n seconds

/base64/:string Returns the decoded :string

Headers

/headers Returns the request headers in 'headers'

/user-agent Returns the user-agent in 'user-agent'

/response-headers Returns a response with headers set from query parameters

Robots

/robots.txt Returns a robots.txt file that disallows access to /deny .

/deny Returns a basic response from a URL that's denied by robots.txt

HTTP Methods

They'll respond with a 404 to non-matching methods

/get Returns a basic response to GET requests

/put Returns a basic response to PUT requests

/post Returns a basic response to POST requests

/patch Returns a basic response to PATCH requests

/delete Returns a basic response to DELETE requests

/gzip Returns gzip-compressed basic request response with 'gzipped': true

Redirects

/redirect/:n Returns a 302 redirect to /redirect/number - 1 if number is greater than 1, otherwise redirects to /get

/relative-redirect/:n Returns a relative redirect to /redirect/number - 1 if number is greater than 1, otherwise redirects to /get

Cookies

/cookies Returns the cookies sent

/cookies/set/:name/:value Returns a response with a Set-Cookie: :name=:value

/cookies/set Returns a response with a set-cookie header constructed from the query parameters

Authentication

/basic-auth/:user/:pass Prompts for authentication and returns 401 unless the provided credentials match :user and :pass

/hidden-basic-auth/:user/:pass Prompts for authentication and returns 401 unless the provided credentials match :user and :pass

/digest-auth Doesn't work
